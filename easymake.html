EasyMake

Simple, lightweight, FOSS alternative to GNU Make or CMake


Gitea | Downloads | Wiki | Discord

What is EasyMake?

Easymake is a brand new buildsystem for all of your C/C++ projects.

We found CMake to be too complicated for new users, and too confusing to migrate old projects over to it, so we took the time to write a brand new, syntax build system.




Download/Source Code

There are pre-build packages on the Gitea repository, or you can compile from source if youd like! There's no libraries or extra dependencies, so it's super portable for all your compiling needs.

The source code can be found on Gitea here.




How To Compile

Its really easy to compile! If you do not use gcc, you can find other compiling options on the wiki.

gcc src/easymake.c -Iinclude -s -O3 -o easymake


Getting started

This is an example of the most basic form of Easymake build file.

More information about build files is given on the wiki here.

{
  "project": "easymake",
  "compiler": "gcc",
  "output": "easymake",
  "includes": ["include"],
  "sources": ["src/*.c"],
  "compiler_options": ["-s", "-O3"]
}
